<h1>Socket 통신 채팅 프로그램</h1>

<br>
<h3>프로젝트 개요</h3>
자바 Socket 통신을 이용하여 여러 명의 이용자가
  <br>
실시간으로 채팅 할 수 있는 채팅 프로그램

<br><br>
<h3>개발 기간</h3>
  2021.06.23 ~ 2021.07.02
  
<br><br>
<h3>개발 환경</h3>
  OS : MacOS
  <br>
  개발툴 : Eclipse
  <br>
  사용언어 : Java
  
<br><br>
